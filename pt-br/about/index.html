About

Hello!

Eu sou Matheus Cunha, um Engenheiro de Software focado em Análise e Engenharia de Sistemas, incluindo DevOps e Engenharia de Dados. Com experiência de campo na arquitetura de sistemas descentralizados e altamente disponíveis, com um histórico demonstrado de trabalho consultivo para muitas empresas, ajudando-as a ser mais future-proof e competitivas em um ambiente altamente mutável. Sempre procurando maneiras de melhorar, aprender mais e ir mais longe.

Também sou amante de tecnologia open-source com skill em Linux, que escreve códigos em:

  • Java;
  • Python;
  • Go;
  • JavaScript (Node.js);
  • e as vezes Lua ou Rust.

Utilizando os seguintes databases:

  • PostgreSQL (para dados relacionais);
  • InfluxDB (incluindo a stack TICK) para time-series;
  • Elasticsearch (incluindo ELK e EBK) para logging e busca;
  • Hadoop (com HDFS, S3 ou Google Cloud Storage) para data warehousing;
  • Kafka como message broker;
  • and Redis para caching e dados efêmeros (as vezes como message broker também).

Monitorando a infrastrutura com Prometheus sendo apoiado pela InfluxData TICK stack (InfluxDB + Kapacitor na maior parte das vezes). Ansible junto com Terraform para fazer infraestrutura como código e de preferência imutável, rodando também Docker em produção dentro do Kubernetes.

Lançando projetos nas principais nuvens: AWS, Azure e GCP, assim como ambientes multi-cloud e nuvem híbrida com servidores on-premises.
